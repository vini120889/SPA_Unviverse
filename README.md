# Projeto SPA Universe 

### Informações do projeto

Nesse projeto foram abordados os seguintes conhecimentos:

- Animações com o CSS;
- Funções no JS;
- Manipulação da DOM;
- Conceitos de SPA;
- Mapeamento de rotas;
- Assíncrono e promises;
- Orientação a objetos;
- Classes;
